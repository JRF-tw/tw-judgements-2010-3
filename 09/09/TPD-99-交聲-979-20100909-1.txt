【裁判字號】    99,交聲,979
【裁判日期】    990909
【裁判案由】    違反道路交通管理處罰條例聲明異議
【裁判全文】
臺灣臺北地方法院交通事件裁定        99年度交聲字第979號
原處分機關  臺北市交通事件裁決所
異  議  人
即受處分人  丙○○
上列異議人即受處分人因違反道路交通管理處罰條例案件，對於
臺北市交通事件裁決所99年5月11日所為之處分（原處分案號：
北市裁罰字第裁22-Z00000000號），聲明異議，本院裁定如下：
    主  文
原處分撤銷。
丙○○汽車駕駛人，駕駛汽車不依規定保持前、後車距離，處罰
鍰新臺幣陸佰元。
    理  由
一、按「汽車行駛於高速公路或設站管制之道路，而不遵管制之
    規定者，處汽車駕駛人新臺幣（下同）3,000元以上6,000元
    以下罰鍰。第1項關於高速公路之使用限制、禁止、行車規
    定及管理等事項之規則，由交通部會同內政部定之」，道路
    交通管理處罰條例第33條第1項、第3項定有明文。次按，高
    速公路交通管制規則第1條規定，本規則依道路交通管理處
    罰條例第33條第3項規定訂定之，足見高速公路交通管制規
    則係依據道路交通管理處罰條例第33條第1項就關於高速公
    路之使用限制、禁止、行車規定及管理等特別事項授權有關
    機關訂定之，是高速公路交通管制規則之於道路交通管理處
    罰條例或道路交通安全規則，係特別法之於普通法之關係。
    果經授權之特別法就某一特殊之情形，有漏未規定者，於該
    一特別法令補充制定完備前，該等情形自有一般規定之適用
    ，此乃法律體系之當然解釋。又按高速公路交通管制規則第
    6條規定，「汽車行駛高速公路，前後兩車間之行車安全距
    離，在正常天候狀況下，依下列規定：
    (一)小型車：車輛速率之每小時公里數值除以2，單位為公尺
      。
    (二)大型車：車輛速率之每小時公里數值減20，單位為公尺。
    前項規定例示如左：
   ┌─────────┬──────────────┐
   │車速（公里／小時）│最小距離（公尺)             │
   │                  ├─────────┬────┤
   │                  │大型車            │小型車  │
   ├─────────┼─────────┼────┤
   │六○              │四○              │三○    │
   │                  │                  │        │
   │七○              │五○              │三五    │
   │                  │                  │        │
   │八○              │六○              │四○    │
   │                  │                  │        │
   │九○              │七○              │四五    │
   │                  │                  │        │
   │一○○            │八○              │五○    │
   │                  │                  │        │
   │一一○            │九○              │五五    │
   └─────────┴─────────┴────┘
    第1項規定如遇濃霧、濃煙、強風、大雨、夜間行車或其他
    特殊狀況時，其安全距離應酌量增加」。然依照上開條文規
    定之行車安全距離，係指在「正常天候狀況」下（第1項）
    ，而附表（第2項），其行車速度又係例示時速60公里至時
    速110公里（即高速公路之行車速度限制），且衡以本條立
    法之目的，乃因考量高速公路行車速度甚快，需有較長之行
    車安全距離，以避免突發之緊急狀況發生，而能即時將車輛
    剎停，顯見本條規定適用之範圍，應係在高速公路正常之天
    候及車流量之狀況下，始有其適用，並非謂凡在高速公路發
    生任何因剎車不及而導致之碰撞均有其適用，此為條文文義
    解釋及目的解釋所當然。是前述高速公路管制規則第6條之
    規定，自屬就時速60公里以下之車道壅塞狀況之安全距離漏
    未規範，然道路交通安全規則第94條第1項規定「汽車在同
    一車道行駛時，除擬超越前車外，後車與前車之間應保持隨
    時可以煞停之距離」，是用路人駕車行駛高速公路，行車時
    速未達60公里時，仍應依上開規定保持前、後車之距離，俾
    可隨時煞停，避免追撞事故。果用路人駕車行駛高速公路，
    因車道壅塞致行車時速未達60公里時，未依上開一般規定保
    持前、後車之距離，自有道路交通管理處罰條例第58條第1
    款，即「汽車駕駛人，駕駛汽車不依規定保持前、後車距離
    ，處600元以上1,200元以下罰鍰」規定之適用。
二、原處分意旨略以：本件異議人即受處分人（下稱受處分人）
    丙○○於民國99年3月23日下午6時56分許，駕駛車牌號碼
    AN-1408號自小客車行經國道一號高速公路北向15.4公里輔
    助車道時，因「未保持行車安全距離致煞車不及追撞車牌號
    碼5982-RJ號自小客車而肇事（無人傷亡）」之違規事實，
    經內政部警政署國道公路警察局第一警察隊依違反道路交通
    管理處罰條例第33條第1項第2款之規定，以公警局交字第Z0
    0000000號違反道路交通事件掣單舉發。嗣將舉發通知單送
    達受處分人後，受處分人於舉發通知單上所載之應到案日期
    即99年4月7日前之99年3月25日向原處分機關提出申訴，經
    原處分機關函請舉發單位即內政部警政署國道公路警察局第
    一警察隊調查結果，仍認受處分人確有前開違規行為，原處
    分機關乃於99年5月11日，依道路交通管理處罰條例第33條
    第1項第2款之規定，裁處受處分人罰鍰3,000元並依同條例
    第63條之規定，記違規點數1點在案。
三、受處分人異議意旨略以：伊於99年3月23日下午6時56分許，
    駕駛車牌號碼AN-1408號自用小客車行至國道一號高速公路
    北向15.4公里輔助車道，準備駛下康寧交流道匝口時，適逢
    下班時段，該處道路呈壅塞狀態，車輛均一部接一部駛駛停
    停，伊依現狀保持距離行駛，惟因煞車不及致追撞前方車牌
    號碼5982-RJ號之自用小客車，經報警處理後，警員遂以伊
    駕駛未保持安全距離為由舉發違規，伊曾向警員說明當時伊
    係依現狀保持距離行駛，追撞肇事係因煞車不及所致，並非
    未保持安全車距，惟未獲接受，是原處分未能依實況裁量酌
    處，實無法令人信服，為此聲明異議，請求撤銷原處分等語
    。
四、經查：
  (一)受處分人於99年3月23日下午6時56分許，駕駛車牌號碼AN-1
    408號自用小客車行經國道一號北上15.4公里輔助車道時，
    因未依規定保持與前車隨時可以煞停之距離，致撞擊前方由
    第三人甲○○駕駛之車牌號碼5982-RJ號自用小客車肇事，
    惟無人傷亡之事實，業據受處分人於本院調查時坦承不諱，
    並有內政部警政署國道公路警察局第一警察隊99年6月21日
    公警一交字第0990102242號函所檢附之道路交通事故調查表
    、現場圖、調查筆錄、現場照片共1份附卷可稽（見本院卷
    第20頁至第40頁），足徵受處分人於前揭時、地確有違反道
    路交通安全規則第94條第1項規定之違規行為甚明。
  (二)又99年3月23日下午6時56分許，國道一號高速公路北向15.4
    公里輔助車道處路況，係呈壅塞現象，車輛走走停停，甚至
    呈停止狀態等情，業據受處分人供述綦詳，核與證人甲○○
    於警詢時證稱：伊當時之車速為零等語（見本院卷第29頁）
    ；證人乙○○於本院調查時證述：「（問：舉發當時該車道
    的車流狀況，是否如本院卷36頁下方現場照片所示，係一部
    接一部車、走走停停的狀況？〈提示本院卷第36頁〉）那個
    交流道幾乎天天都這樣子，特別是上下班時段會這樣子」等
    語相符（見本院卷第45頁反面），並有現場照片1張存卷足
    參（見本院卷第36頁下方照片），堪認本件事故發生時，上
    揭路段已呈非正常之壅塞狀況，致行車時速無法達60公里，
    而無高速公路管制規則第6條規定之適用，然如前述，高速
    公路管制規則既係道路交通管理處罰條例第33條第1項所授
    權訂定之法規，則其所未規定之事項自應回歸適用授權母法
    之規定。是本件受處分人既有上開「駕駛汽車不依規定保持
    前、後車隨時可以煞停距離」之違規行為，自應依道路交通
    管理處罰條例第58條第1款之規定裁罰，方屬適法。
五、綜上所述，本件並無高速公路管制規則第6條規定之適用，
    原處分機關依道路交通管理處罰條例第33條第1項2款之規定
    ，裁處受處分人罰鍰3,000元並依同條例第63條之規定記違
    規點數1點，於法顯有未洽，受處分人此部分之異議為有理
    由，然因受處分人仍有上開「駕駛汽車不依規定保持前、後
    車距離」之違規行為，自應由本院將原處分撤銷，並依道路
    交通管理處罰條例第58條第1款及違反道路交通管理事件統
    一裁罰標準之規定，裁處受處分人罰鍰600元，如主文第2項
    所示，以資適法。
據上論斷，應依道路交通案件處理辦法第20條前段，裁定如主文
。

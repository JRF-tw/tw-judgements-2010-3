【裁判字號】    99,簡上,355
【裁判日期】    990720
【裁判案由】    詐欺
【裁判全文】
臺灣板橋地方法院刑事判決             99年度簡上字第355號
上  訴  人
即  被  告  乙○○
上列上訴人因詐欺案件，不服本院99年度簡字第257 號中華民國
99年2 月24日第一審判決（聲請案號：98年度偵字第30853 號）
，提起上訴，本院管轄之第二審合議庭判決如下：
    主  文
上訴駁回。
    事實及理由
一、本案經本院審理結果，認第一審簡易判決認被告乙○○犯刑
    法第30條、第339 條第1 項之幫助詐欺取財罪，判處拘役二
    十日，如易科罰金以新臺幣一千元折算一日，認事用法及量
    刑均無不當，應予維持，除原審判決所認被告交付本件銀行
    帳戶金融卡及密碼時間，應更正為原檢察官聲請簡易判決處
    刑所認之「民國98年10月15日下午3 時許」外，其餘均引用
    如附件第一審判決書記載之事實、證據及理由。
二、被告上訴意旨略以：被告係因見報登應徵司機廣告，與對方
    聯絡，對方要求其提供本件銀行帳戶之金融卡，供對方做信
    用調查，被告係遭詐騙利用，其本人並無幫助他人犯本件詐
    欺罪之故意，爰提起上訴云云。
三、但查，原審判決所認被告乙○○幫助詐欺取財之犯罪事實，
    業據被害人甲○○於警詢指述綦詳（見偵查卷11至12頁，此
    被害人於警詢之陳述，被告就其證據能力未到庭爭執，依刑
    事訴訟法第159 條之5 規定，得為證據），並有被告本件上
    開銀行帳戶開戶資料、98年存摺存款歷史明細批次查詢資料
    、被害人甲○○被騙匯款之郵政自動櫃員機交易明細表、存
    摺明細、報案之新竹市警察局第三分局青草湖派出所受理刑
    事案件報案三聯單、受理詐騙帳戶通報警示簡便格式表、金
    融機構聯防機制通報單、內政部警政署反詐騙諮詢專線紀錄
    表、被告持用之行動電話門號0000000000號雙向通聯紀錄等
    資料附卷可資佐證。被告上訴意旨雖矢口否認上揭幫助詐欺
    犯行，辯稱上開意旨云云，然查：(一)被告係有多年社會工作
    經驗之成年人，此有被告勞工保險投保資料在卷可稽，且其
    於本院準備程序時亦自陳：伊之前應徵工作之公司從無要伊
    提供金融卡、存摺供公司調查信用之情等語明確（見本院99
    年5 月6 日準備程序筆錄3頁 ），豈有在外隨意相約地點而
    尚不知對方公司所在地及實際營運狀況下，即貿然將其個人
    重要之帳戶金融卡交付不詳之人支配使用，此已與常情有違
    ，且其上開辯稱對方要求其交付本件銀行帳戶金融卡係為查
    核其個人信用情形云云，不僅與社會一般對應徵者調查個人
    信用方式有異，再者被告係提供其個人內僅有零頭金額存款
    之帳戶，如何供作調查信用使用，況被告另辯稱其並無主動
    提供本件銀行帳戶金融卡及密碼給對方，而係於對方與其聊
    天時，其在全然不知之情況下將其本件銀行帳戶金融卡及密
    碼交付給對方云云（見本院上開準備程序筆錄2 至3 頁），
    復見其所辯情節前後矛盾不一，亦與常理不合，匪夷所思，
    徵諸上述疑點，亦見被告上開所辯情節，顯屬無稽。(二)次查
    ，被告於警詢時原係辯稱：伊於98年10月14日要應徵兼職司
    機工作，該公司員工稱要測試伊信用狀況，叫伊拿伊所有之
    提款卡給他測試，如果可以的話，同年月19日會通知伊上班
    ，直到約定之19日，對方都沒有與伊聯絡，伊打對方電話，
    但對方電話已暫停使用，伊當時發覺有異，即至三重市○○
    路上一家臺灣銀行辦理提款卡掛失，該銀行行員告知伊該帳
    戶已遭凍結列為警示帳戶無法使用，要伊自行至派出所尋求
    警方協助，因此伊於今日（即同年月25日）自行至派出所製
    作筆錄接受調查，伊該帳戶是作為一般存款使用云云（見偵
    查卷5 至6 頁），然核諸被告其後於原審提出之本件報紙徵
    才廣告、其以0000000000行動電話門號與對方之0000000000
    、0000000000等行動電話門號通話之明細單所示，被告係於
    98年10月15日見報徵才廣告後，而於98年10月15日上午11時
    52分許先撥打電話與對方聯絡，其後於98年10月15日下午5
    時56分許、98年10月16日下午2 時8 分許、5 時44分許及98
    年10月17日下午3 時28分許接聽對方撥打與其聯絡之電話，
    此亦有本院調閱之被告上開行動電話門號之雙向通聯記錄在
    卷可按，已見被告上開警詢所辯見報聯絡應徵時間、通話情
    節等，顯均與事實不符。而其後於偵查中又辯稱：伊於98年
    10月14日打電話求職，對方約伊於98年10月15日準備身分證
    、駕照及空白帳戶金融卡，他說空白帳戶是用來薪資轉帳，
    之後伊與對方約在板橋市○○○路附近將東西交給對方，伊
    交付時對方拿煙請伊抽，伊抽後頭就暈，並告知對方伊金融
    卡之密碼，對方並說98年10月17日、18日公司辦旅遊，要伊
    於98年10月19日到公司上班，之後伊就無法聯絡上對方，且
    對方電話變成空號，伊該帳戶是用來存款及買黃金使用云云
    （見偵查卷26至27頁），亦見與其上開警詢陳述情節未盡相
    符，亦與上開其提出之報紙徵才廣告及通聯記錄所示不合，
    況其上開所稱交付對方本件帳戶金融卡及密碼情節，亦屬玄
    虛，顯與一般事理相悖，難以置信。又被告其後於原審及本
    院審理時雖又具狀或到庭改詞陳稱：伊係於98年10月15日上
    午約11時許，見報紙徵才廣告打電話與對方一位自稱「王先
    生」之人聯絡，對方約伊於當天下午6 時許，在板橋文化路
    與漢生東路交接處見面，請伊準備身分證、駕照、存摺、金
    融卡，存摺是要薪資轉帳使用，金融卡則是要查詢伊信用狀
    況，至當日將近下午6 時許，伊又接獲「王先生」來電，將
    相約地點改在板橋漢生東路騎樓下，之後是一位自稱「林經
    理」來面試，伊在「林經理」與伊面試聊天中，就在不知情
    下將伊本件銀行帳戶存摺、提款卡、身分證影本交給對方，
    並告訴他密碼，然後告訴伊回去等通知，如何可以將請伊於
    隔天晚上7 時上班，至次日即同年月16日下午2 時許，「林
    經理」通知伊當天晚上7 時可上班，但至同日下午5 時許，
    他又來電稱因公司忙，請伊於隔天即同年月17日再去上班，
    但至次日即同年月17日下午3 時許，「林經理」又來電稱因
    公司這兩日員工旅遊，要伊於同年月19日再去上班，但至同
    年月19日因「林經理」未與伊聯絡，伊打電話欲與他聯絡即
    無法聯絡，至同年月21日因伊回三重客運公司辦理離職手續
    ，要拿身分證時，才發現伊上開帳戶金融卡不見，因伊忘記
    依該金融卡如何不見，至銀行要報遺失，銀行人員告知伊該
    帳戶已列為警示帳戶，伊才去警局報案云云（見原審99年度
    簡字第257 號刑事卷12至14頁、本院上開準備程序筆錄2 至
    3 頁），惟核諸被告上開所辯情節，與其上開警詢、偵查所
    述前後不一，是否屬實顯堪質疑，且其所稱於同年月21日至
    三重客運公司辦理離職，始發現本件帳戶金融卡不見，因忘
    記為何不見，所以至銀行報遺失等情云云，然核此距其交付
    上開金融卡僅有6 天，且其甫於2 天前（即同年月19日）未
    能與對方聯絡而無法上班，被告上開辯詞要與常理不合且屬
    矛盾，不免有因虛辯而無法自圓之嫌；至其上開所述與對方
    電話聯絡時間，雖與其上開行動電話門號通聯紀錄相合，惟
    衡諸與其前於警詢、偵查所述不合之情，亦顯有事後依上開
    通聯紀錄所示，配合編詞飾卸之虞，尚難逕予採信。(三)另稽
    之被告於交付本件帳戶金融卡前，即先於98年10月15日下午
    1 時34分許，將該帳戶存款提款新臺幣（下同）4000元，僅
    餘存款27元，有上開該帳戶存摺存款歷史明細批次查詢資料
    在卷可按，衡諸此舉與一般交付帳戶供他人使用前，預先清
    空該帳戶存款之情相類，且依被告上開所辯情節，其早於98
    年10月15日即與對方失聯，卻未就其已交付對方銀行金融卡
    掛失或報警處理，縱依其上開所述其最遲至同年月21日，始
    發現本件上開金融卡不見，而其卻又遲滯於同年月25日始至
    警局報案製作筆錄，此有偵查卷附之被告警詢筆錄、本院99
    年5 月28日辦理刑事案件電話記錄查詢表在卷可參，亦見其
    上述所為有違常情，是其所辯難以採信。綜上所述，足見被
    告上開所辯，應係卸責之詞，不足採信，且參互印證上述理
    由事證，堪認本件上開銀行帳戶之金融卡，應係被告交付予
    詐欺集團供作詐騙被害人匯款之人頭帳戶使用無訛。是本件
    事證已臻明確，被告上揭犯行洵堪認定。從而，本件原審判
    決認事用法，核無不合，另衡諸被告本件犯罪動機、手段、
    所生危害及犯後態度等一切情狀，原審判決量刑亦屬妥適。
    而被告仍執上詞提起上訴，顯無理由，自應予駁回。
據上論斷，應依刑事訴訟法第455 條之1 第1 項、第3 項、第36
8 條、第373 條，判決如主文。

附錄本案論罪科刑法條全文：刑法第339條
中華民國刑法第339條
（普通詐欺罪）
意圖為自己或第三人不法之所有，以詐術使人將本人或第三人之
物交付者，處 5 年以下有期徒刑、拘役或科或併科 1 千元以下
罰金。
以前項方法得財產上不法之利益或使第三人得之者，亦同。
前二項之未遂犯罰之。

│    事實及理由                                              │
│一、本件犯罪事實、證據，除犯罪事實欄倒數第3 、4 行「於民    │

│二、刑法上之幫助犯，係以幫助之意思，對於正犯資以助力，而    │
│    未參與實施構成要件之行為者而言。本件被告將其所有之帳    │
│    戶提款卡及密碼，交付予某真實姓名年籍不詳之詐欺成員使    │
│    用，雖使該成員得基於詐欺取財之犯意，向被害人甲○○施    │
│    用詐術，而遂行詐欺集團成員詐欺取財之犯行，惟被告單純    │
│    提供帳戶提款卡供人使用之行為，並不等同於向被害人施以    │
│    欺罔之詐術行為，且亦無證據證明被告有參與詐欺取財犯行    │
│    之構成要件行為。是核被告所為係犯刑法第30條第1 項前段    │
│    、第339 條第1 項之幫助詐欺取財罪。被告係基於幫助之犯    │
│    意，而並未實際參與詐欺犯行，所犯情節較正犯輕微，為幫    │
│    助犯，酌依刑法第30條第2 項規定減輕其刑。爰審酌被告雖    │
│    未實際參與詐欺取財犯行，但其提供帳戶提款卡及密碼供不    │
│    法犯罪集團使用，不僅造成執法機關不易查緝犯罪行為人，    │
│    嚴重危害社會治安，助長社會犯罪風氣，更造成被害人求償    │
│    上之困難，兼衡被告之犯罪後態度等一切情狀，量處如主文    │
│    所示之刑，並諭知易科罰金之折算標準。                    │

│    取得上開帳戶後，意圖為自己不法之所有，於98年10月16日    │
│    下午6時許，假冒臺灣銀行客服人員，撥打電話予甲○○，     │
│    向其佯稱網路購物付款設定錯誤，要至自動櫃員機取消分期    │
│    付款設定，致甲○○陷於錯誤，信以為真，遂依其指示，於    │
│    同日晚上7時許，至新竹市○區○○路621號之郵局匯款新臺    │
│    幣2萬4538元，至乙○○所提供之上開帳戶，該犯罪集團成     │
│    員即以此方式詐騙財物得手。嗣甲○○轉帳後發覺有異報警    │
│    處理，始悉上情。                                        │
│二、案經臺北縣政府警察局三重分局移送偵辦。                  │
│         證據並所犯法條                                     │
│一、訊據被告乙○○雖承認上開帳戶係其所有，惟矢口否認有任    │
│    何詐欺犯行，辯稱：伊看報紙應徵晚班司機工作，對方說要    │
│    測試伊之信用，所以要伊交付提款卡，伊才會將帳戶之提款    │
│    卡及密碼交給該真實姓名年籍不詳之人云云。經查：被告江    │
│    新達之上開帳戶遭詐欺集團成員用以行騙之事實，業據被害    │
│    人甲○○指述稽詳，並有臺灣銀行南新莊分行存摺資料、被    │
│    害人匯款明細及其臺灣銀行帳戶存摺資料等附卷可參。雖被    │
│    告以其帳戶提款卡及密碼係交付求職之公司為抗辯，惟金融    │
│    機構帳戶係個人理財之重要工具，且關係個人財產、信用之    │
│    表徵，豈能隨意交予陌生人，如該他人非供作犯罪所得出入    │
│    之帳戶或其他不法目的，應無使用非自己名義金融帳戶之必    │
│    要；又現今利用電話而詐欺取財之犯罪類型層出不窮，該等    │
│    犯罪多數均係利用人頭帳戶作為出入帳戶，並經媒體廣為披    │
│    載，被告要難諉為不知，足認被告有容任不詳之詐欺集團人    │
│    員共犯利用該帳戶犯詐欺取財罪之不確定故意，允無疑義；    │
│    又被告雖辯稱欲求職才交付提款卡及密碼，惟其無法提供求    │
│    職時之報紙、聯絡電話，甚連求職公司之名稱皆不知，被告    │
│    是否真為求職而交付上開帳戶之提款卡及密碼，非無疑義；    │
│    另該不詳之詐欺集團果利用被告交付之帳戶作為犯詐欺取財    │
│    罪之款項出入帳戶，被告犯嫌足堪認定。                    │
│二、按所謂幫助行為乃指對於被幫助者之犯罪行為，予以物質或    │
│    精神上之支持，而使其得以或易於實現構成要件，或使其行    │
│    為造成更大之損害。準此，幫助犯之幫助行為應係指實施構    │
│    成要件以外之行為，若係實施構成要件之行為，則非幫助行    │
│    為，而應論以共同正犯。經查，本件並無證據顯示被告江新    │
│    達為該詐欺集團之成員，雖甲○○確實匯款至被告前開提供    │
│    之帳戶，業如前述，但既無從證明本件詐欺罪之構成要件行    │
│    為或該匯入之款項，為被告所實施或提領，故僅能認定被告    │
│    係單純提供帳戶供人使用，而為幫助詐欺取財之行為。是核    │
│    被告所為，係犯刑法第30條第1項前段、第339條第1項之幫     │
│    助詐欺取財罪嫌。                                        │
